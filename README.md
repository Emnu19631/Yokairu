# mi_juego

## Overview
This project is a simple game developed in Python. It includes various components such as game logic, a menu system, and asset management.

## Project Structure
```
mi_juego/
├── main.py          # Main entry point for the game
├── src/             # Source files for game logic
│   ├── __init__.py  # Initializes the src package
│   ├── juego.py     # Core game logic
│   ├── menu.py      # Menu system
│   └── guardado.py  # Save and load game data
├── assets/          # Game assets
│   ├── img/         # Image assets
│   ├── sound/       # Sound assets
│   └── fonts/       # Font files
├── saves/           # Saved game data
│   └── savegame.json
├── requirements.txt  # Dependencies for the project
└── README.md        # Project documentation
```

## Installation
To install the required dependencies, run the following command:

```
pip install -r requirements.txt
```

## Running the Game
To start the game, execute the main.py file:

```
python main.py
```
